JetStore Digital
Descripción

JetStore Digital es una plataforma web orientada a la comercialización de libros digitales en formato PDF. El sistema permite a los usuarios explorar un catálogo de productos digitales, realizar compras y acceder a contenido educativo de manera rápida y segura.

Tecnologías Utilizadas
Frontend
Next.js
React
TypeScript
Tailwind CSS
Backend
Spring Boot
Java
Maven
API REST
Base de Datos
MySQL
Instalación del Proyecto
1. Clonar el Repositorio
git clone https://github.com/USUARIO/jetstore-digital.git

Ingresar a la carpeta del proyecto:

cd jetstore-digital
2. Ejecutar el Frontend

Ingresar a la carpeta frontend:

cd frontend

Instalar dependencias:

npm install

Ejecutar el servidor de desarrollo:

npm run dev

Abrir en el navegador:

http://localhost:3000
3. Ejecutar el Backend

Abrir una nueva terminal.

Ingresar a la carpeta backend:

cd backend

Compilar el proyecto:

mvn clean install

Ejecutar Spring Boot:

mvn spring-boot:run

El backend estará disponible en:

http://localhost:8080
